About Kcaesar-Cipher

Kcaesar's cipher is just one of many cryptography techniques used for fun .The cool thing about this cipher is that it look like morse code to fool others but one with any idea about kcaesar's cipher can tell the diffrence as it uses different methods that doesn't exist in morse code

To use this cipher we have to convertthe alphabets to their numerical value that is A-Z = 1-26

then just convert the numbers to binary format. After converting the numbers into binary we then convert the binary into (.) and (-) where 0 is (.) 1 is (-)

Example

L --> 12 --> 1100 --> --..
